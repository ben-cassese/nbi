nbi
=====

**Neural Bayesian Inference for 1D Sequences**

To use this package

```bash
git clone https://github.com/kmzzhang/nbi
cd nbi
pip install -e .
```

The -e option for pip is the development mode, where changes will immediately apply

For Jupyter notebooks, set
```jupyter
%load_ext autoreload
%autoreload 2
```
